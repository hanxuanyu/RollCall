# RollCall 课堂点名

RollCall 是一个面向课堂使用的随机点名与积分管理工具。教师可以维护班级和学生名单，在课堂上按指定人数随机抽取学生，并在点名结果中直接加减积分。

程序支持两种使用方式：

- 桌面应用：适合单机、离线、投屏或触控屏使用。
- 浏览器访问：适合同一台电脑或局域网内通过浏览器使用。

## 主要功能

- 随机点名：支持一次抽取 1 到 10 名学生。
- 点名动画：提供课堂展示用的动态抽取效果。
- 多班级管理：可创建多个班级，并设置默认班级。
- 学生名单管理：支持手动添加、编辑、禁用、删除、搜索和排序。
- 批量导入导出：支持 CSV、Excel 文件，也支持直接粘贴文本导入。
- 积分管理：支持单人加减分、批量加减分、手动调整积分和撤销积分记录。
- 点名策略设置：支持公平模式、纯随机模式、积分权重模式。
- 后台密码：可为管理后台设置访问密码。

## 快速开始

1. 启动程序。
2. 首次使用时，如果还没有班级，点击“进入管理后台”。
3. 在“班级”页面创建班级，并可将常用班级设为默认班级。
4. 进入“学生”页面，手动添加学生，或通过导入功能批量录入名单。
5. 点击“返回点名”，回到点名主界面。
6. 选择班级、点名模式和抽取人数，点击“开始点名”。
7. 点名结束后，在结果弹窗中查看被抽中的学生，并可直接加减积分。

## 点名主界面

点名页是课堂使用的主界面，包含以下操作：

- 班级选择：切换当前点名班级。
- 总人数：显示当前班级可参与点名的正常状态学生数量。
- 模式选择：切换点名策略。
- 抽取人数：选择本次要抽取的人数，最多 10 人。
- 开始点名：启动抽取动画并生成结果。
- 后台入口：进入班级、学生、积分和设置管理页面。

点名结果弹出后，可以对单个被抽中的学生加分或扣分，也可以对本次抽中的所有学生统一加分或扣分。积分操作会写入积分记录，便于后续查看和撤销。

## 点名模式

- 公平模式：最近被抽中过的学生权重降低，长期看更均衡。推荐日常课堂使用。
- 纯随机：所有可用学生概率相同。
- 积分权重：积分较低的学生被抽中概率更高，同时仍会结合最近点名记录降低重复概率。

可以在点名页直接切换模式，也可以在“设置”页面保存为长期配置。

## 班级管理

在管理后台的“班级”页面可以：

- 新建班级。
- 编辑班级名称。
- 删除班级。
- 设置默认班级。
- 查看每个班级的学生数量。

删除班级会同时删除该班级下的学生数据和相关记录，执行前请确认数据已不再需要。

## 学生管理

在“学生”页面可以维护当前班级的学生名单：

- 添加学生：填写姓名、学号和性别。
- 编辑学生：修改姓名、学号、性别和状态。
- 禁用学生：禁用后不参与点名，但数据仍保留。
- 删除学生：移除学生记录。
- 搜索学生：按姓名或学号搜索。
- 排序分页：按学号、姓名、性别、积分、状态排序，并设置每页显示数量。

建议使用“禁用”处理短期不参与点名的学生，只有确认不再需要该记录时再删除。

## 批量导入学生

“学生”页面点击“导入”后，可以选择文件导入或文本粘贴。

支持格式：

- `.csv`
- `.xlsx`
- `.xls`
- 直接粘贴逗号、制表符、分号或竖线分隔的文本

推荐列顺序：

```text
姓名,学号,性别,积分
张三,2024001,男,0
李四,2024002,女,0
```

其中姓名必填，学号、性别、积分可选。

导入前会显示预览表格，可在预览中继续编辑、删除或新增行。遇到同名同学号的重复学生时，可以选择跳过重复项或覆盖已有数据。

## 导出学生

在“学生”页面点击“导出”即可导出当前班级学生名单。导出内容包含姓名、学号、性别、积分和状态。

## 积分管理

在“积分”页面可以：

- 查看当前班级正常状态学生列表。
- 点击学生行选择多个学生。
- 对单个学生快速 `+1` 或 `-1`。
- 直接点击积分数值，手动输入目标积分。
- 对选中的学生批量加减分，并填写原因。
- 查看积分记录。
- 撤销某条积分记录。
- 按积分排序，快速查看高分或低分学生。

积分可以为负数。每次加减分都会生成记录，撤销记录会反向恢复对应积分。

## 系统设置

在“设置”页面可以调整：

- 点名模式：公平模式、纯随机、积分权重。
- 防重复窗口：最近 N 次点名中被抽中过的学生会降低权重，填 0 表示不限制。
- 积分系统：启用或关闭积分功能。
- 动画效果：启用或关闭点名动画。
- 动画时长：设置点名动画持续时间。
- 操作栏位置：选择顶部或底部操作栏。
- 点名记录：清空当前班级抽取记录，重新计算防重复和权重。
- 权重数据：查看当前班级每个学生的权重和抽中概率。
- 后台密码：设置、修改或清除管理后台密码。

设置完成后点击“保存设置”生效。

## 数据保存位置

桌面应用：

- macOS：`~/Library/Application Support/RollCall`
- 其他系统：应用可执行文件所在目录

浏览器 Server 模式：

- 配置目录：`./config`
- 数据目录：`./data`
- 数据库文件：`./data/rollcall.db`

建议定期备份 `data` 目录，尤其是在批量导入、删除班级或清空记录之前。

## 常见问题

没有学生参与点名：

- 确认已经选择班级。
- 确认该班级下有学生。
- 确认学生状态为“正常”，被禁用的学生不会参与点名。

点名经常抽到同一批学生：

- 使用“公平模式”。
- 在设置中适当增大“防重复窗口”。
- 检查是否清空过点名记录。

导入失败或数据为空：

- 确认第一列是姓名。
- 确认文件格式为 CSV、XLSX 或 XLS。
- 如果使用文本粘贴，确保每行一名学生。
- 导入预览中空姓名行会被忽略。

忘记后台密码：

- 后台密码保存在配置文件中。若无法进入后台，需要由管理员处理配置文件中的密码字段。
